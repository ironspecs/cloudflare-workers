Local Example

Open the real newsletter signup dialog.

This page is the working example for the public window.Newsletters.open(...) API. When served from localhost or 127.0.0.1, it uses the Turnstile test key path and the submission is intentionally sunk instead of stored.

Use this page for local smoke testing. On a real approved hostname, the same API path writes live subscriptions.

	window.Newsletters.open({ listName: 'weekly' });
});
